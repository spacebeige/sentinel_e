# Sentinel-E

Sentinel-E is a dual-scope AI safety and risk boundary analysis system.

## Features
- Standard and experimental analysis modes
- Multi-model hypothesis extraction and debate
- Boundary violation detection and feedback
- FastAPI backend and React frontend

## Getting Started
1. Clone the repository
2. Set up Python and Node.js environments
3. Run the backend with FastAPI (see backend/main.py)
4. Run the frontend (see frontend/README or package.json)
